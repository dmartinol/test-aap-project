# CVE Remediation Playbooks

Ansible playbooks for CVE remediation on Red Hat Enterprise Linux systems.

## Structure

- `playbooks/remediation/` - CVE remediation playbooks
- `playbooks/roles/` - Shared Ansible roles (future)
- `inventories/` - Inventory files (future)

## Usage

These playbooks are designed to be executed via Ansible Automation Platform (AAP).

## Requirements

- Ansible 2.9+
- Red Hat Enterprise Linux target systems
- Appropriate credentials and permissions for package management
